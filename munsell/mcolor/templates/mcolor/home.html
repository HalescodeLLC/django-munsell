Munsell Color Conversion For Everyone!

What does this do?

Given all or part of a Munsell Color, Munverter will return the corresponding name, RGB and hexadecimal values. Munverter contains mostly soil colors at this point, making it particularly useful to Archaeologists, Soil Scientists, etc. The eventual goal is to create a complete reference system, but we aren't there yet. If you find a bug or think of something else you'd like it to do, you can contact us.

How do I use it?

All you have to do is type your munsell color in the box up top, then click on one of the match buttons. Here are a couple of entry examples. Although if you click the Exact Match button only the first two will return anything. More on that later.

  • 5yr 4/6
  • N 5
  • 5y
  • n

What's the difference between Fuzzy and Exact Matches?

The Fuzzy Match button will return every munsell color that contains whatever you've entered in the search box. The Exact Match button is far pickier and will only return perfect matches. Try it out and it'll make sense.

Why didn't it work?

If you're sure you've entered a valid munsell color (and not the word 'hamster' or something) but no results were returned, we probably haven't added that color yet. If you want us to add your color, simply send us the Munsell Color (5YR 5/4) and it's offical name ('Reddish Brown') and we'll take care of the rest!
